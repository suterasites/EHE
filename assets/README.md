## Brand Identity (from logo + existing site)

**Logo:** EHE roundel - bold orange + black "EHE" lettering on white, hexagonal shield outline, "EARTH HAUL / EXCAVATIONS" wordmark in chunky military/industrial sans-serif. The brand is rugged, traditional tradie, not modern/minimal.

**Palette (sourced from logo):**
- Primary accent: `#F26522` (EHE orange) - CTAs, link hover, accent strokes
- Dark: `#1A1A1A` (near-black) - nav, dark sections, body text on light
- Mid: `#3D3D3D` (charcoal) - secondary surfaces
- Light: `#FFFFFF` and `#F4F4F4` - body backgrounds
- Optional warm earth tone: `#8B6B47` if a dirt-toned accent is needed

**Typography intent:**
- Display / H1: chunky condensed sans-serif (Oswald / Bebas Neue / Antonio) to echo logo wordmark
- Body: clean, legible sans-serif (Inter / IBM Plex Sans)
- Avoid: rounded geometric sans-serifs (out of character)

**Voice / register:**
- Aussie tradie / owner-operator (per `feedback_aussie_tradie_voice`).
- Lead with outcomes, not features (per `feedback_sell_outcomes_not_features`).
- Single-operator business - copy must say "I" not "we" where personal (Braeden runs the gear himself per existing site).

**Icon usage (per James):**
- Icons ONLY for phone, email, social. No decorative service-tile icons, no abstract feature icons, no checkmark bullets with icon flourishes. The service tiles use photos, not icons.
