# Earth Haul Excavations - Mockup

Free homepage mockup for Earth Haul Excavations (Braeden McCarthy, Benalla VIC) - built by Sutera Sites.

## Status
Pre-launch mockup. Single static page (`index.html`) with assets, CSS, JS.

## Deployment
Cloudflare Pages preview, deployed from GitHub repo `suterasites/EHE`.

## Stack
- Static HTML/CSS/JS (no build step)
- Oswald + Inter via Google Fonts
- No JS dependencies

## Formspree
The contact form action attribute is a placeholder (`REPLACE_WITH_FORMSPREE_ID`). Wire in the real endpoint before the prospect interacts with the live URL.

## Brand
Orange `#E85A1F`, near-black `#1A1A1A`, white. Oswald display, Inter body. Industrial tradie / owner-operator.
